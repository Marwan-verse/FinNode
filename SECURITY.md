# Security Policy

## Supported Versions

Security updates are provided for the active development line.

| Version | Supported |
|---|---|
| 0.2.x | Yes |
| < 0.2.0 | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Use one of these methods:

1. GitHub Security Advisories:
   - https://github.com/Marwan-verse/FinNode/security/advisories/new
2. Private email (if maintained by your team):
   - `security@finnode.local` (replace with your real address)

When reporting, include:

- Affected version and platform
- Reproduction steps
- Expected vs actual behavior
- Potential impact
- Suggested mitigation (if any)

## Response Expectations

- Initial triage target: within 5 business days
- Fix target: depends on severity and exploitability
- Coordinated disclosure preferred for high/critical issues
